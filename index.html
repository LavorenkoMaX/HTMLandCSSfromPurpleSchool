Під час розробки додатка періодично потрібно оперувати фіксованим набором рядкових значень. Це може бути що завгодно. Наприклад, список колірних кодів, що підтримуються додатком, або список пристроїв, з якими здатний взаємодіяти наш додаток, пори року тощо.

"Строкові константи"

Все це — заздалегідь визначений набір рядкових констант із загальною структурою, з якими необхідно налагодити програмну взаємодію на рівні Java-коду. Коли необхідно взаємодіяти із заздалегідь визначеним набором рядкових (і не тільки) констант, найкраще рішення — написати свій enum-клас. Нижче розглянемо приклади перетворення enum у string. Створення рядкових переліків Створимо клас enum, в якому зберігається список різних середовищ для розгортання додатка, а також URL кожного середовища:

У даному класі ми визначили 4 середовища: PROD — релізне TEST — для ручного тестування AUTO_TEST — середовище для автотестів DEV — локальне для розробки та дебагування А також 4 URL для кожного з цих середовищ.

Звернемо увагу на деякі важливі моменти. Кожен URL — це рядкова константа нашого переліку: вони визначаються в дужках біля кожної константи enum. Обов'язкова наявність конструктора, що приймає аргумент того ж типу, що і кожна enum константа. Область видимості конструктора — private або package private. Необхідно визначити змінну — поле класу, яке буде зберігати визначені нами рядкові константи. Для цього поля необхідно створити метод геттер, щоб використовувати значення рядкових констант ззовні.

Перебір рядкових перерахувань

На цьому етапі ми можемо перебрати всі доступні enumзначення, і навіть отримати прив'язані до них рядкові константи. Щоб отримати всі значення будь-якого enumкласу, необхідно використовувати метод values():
